Overview
This project is an online grocery store system designed to simulate a real-world e-commerce experience.
Users can explore products by category, add them to a cart, update quantities, and place orders.

The application focuses on usability, responsive design, and practical implementation of front-end and back-end integration.

Features
- Browse products by category and subcategory
- Search products with keyword suggestions
- Add items to the shopping cart
- Update item quantity or remove items
- Cart persistence using localStorage (20-minute session)
- Disable purchase when items are out of stock
- Order processing and tracking
- Responsive UI with interactive product highlighting
